# **App Name**: CyberFolio

## Core Features:

- Interactive Project Showcase: Display user's projects with titles, detailed descriptions, and embedded media (images, videos, or gifs), with interactive elements reminiscent of the Watch Dogs 2 UI.
- About & Skills Matrix: A dedicated section detailing the user's professional journey, skills, and resume, presented in a dynamic, 'hacker'-style interface.
- Secure Contact Module: A visually distinct contact section or form for visitors to connect, including links to social profiles and email, designed with the game's UI aesthetics.
- Thematic UI Elements: Implement custom UI components, subtle digital animations, and transitions that emulate the distinct 'hacker' aesthetic and interactive feel of Watch Dogs 2.
- Content Configuration: Allow for easy updating of all portfolio content (projects, skills, about me) through simple, locally managed configuration files (e.g., JSON or Markdown), suitable for an MVP.
- AI Project Synopsis Tool: An optional tool where the user can input project details and receive AI-generated summary synopses in various tones or styles, designed to enhance project descriptions.

## Style Guidelines:

- Color Scheme: Dark, emphasizing a futuristic, tech-noir aesthetic.
- Primary color: A vibrant cyber-purple (#8B4DF2) for interactive elements and key highlights.
- Background color: A very dark, subtle purple-black (#121013) to create depth and contrast.
- Accent color: An electric blue (#62A2F7) to complement the primary, used for secondary actions or information display.
- Headline font: 'Hacked' (decorative display font) for titles and section headings. Note: currently only Google Fonts are supported.
- Body font: 'Inter' (sans-serif) for general text, ensuring readability within the tech-themed interface.
- Geometric, minimalistic icons, primarily monochromatic or duotone, with subtle glow effects and vector crispness, aligning with the Watch Dogs 2 UI style.
- Modular, grid-based layouts reminiscent of a futuristic HUD (Heads-Up Display), with clear data segmentation and ample negative space to enhance focus on content.
- Integrate glitch effects, swift digital transitions, and subtle holographic animations for navigation and content loading, enhancing the immersive hacker aesthetic.
